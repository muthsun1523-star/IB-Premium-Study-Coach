# IB-Premium-Study-Coach
AI-powered Progressive Web App for IB MYP (Grade 10) and DP (Grades 11-12) students. Features personalized AI tutoring, homework review with IB rubric feedback, smart study planning, and comprehensive progress tracking
## Features

### 🎓 AI Tutoring
- **Four tutoring modes**: MYP Conceptual, DP Exam-Ready, Quick Revision, Deep Dive
- **Voice interaction**: Real-time speech-to-text and text-to-speech
- **Streaming responses**: SSE-based streaming for instant feedback
- **Context-aware**: Adapts to student's programme, grade, and subject

### 📝 Homework Review
- **Multi-file upload**: PDF, images, DOCX, TXT support
- **AI feedback**: Detailed analysis aligned with IB assessment criteria
- **MYP criteria scoring**: Levels 0-8 for criteria A, B, C, D
- **Command term analysis**: Identifies and suggests proper command term usage

### 📊 Assessments
- **Multiple question types**: MCQ, short answer, extended response
- **Auto-grading**: Instant feedback with explanations
- **Diagnostic tests**: Identify knowledge gaps
- **Mastery tracking**: Spaced repetition algorithm

### 📅 Study Planning
- **AI-generated plans**: Based on exam dates and available time
- **Calendar integration**: Weekly and daily views
- **Spaced repetition**: Smart review scheduling
- **Progress tracking**: Visual completion stats

### 💳 Subscription System
- **3-day free trial**: Full access to all features
- **Razorpay integration**: Secure payment processing
- **Webhook handling**: Real-time subscription updates
- **Grace period**: 48-hour buffer for payment issues
